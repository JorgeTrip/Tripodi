# Tripodi — Viaje al Origen de un Apellido Milenario

Un sitio web interactivo que documenta la historia, etimología y diáspora del apellido Tripodi, explorando sus raíces en la antigua Grecia y su evolución a través de los siglos.

## 📋 Descripción del Proyecto

Este proyecto es una página web educativa y cultural que presenta una investigación exhaustiva sobre el apellido Tripodi. A través de una interfaz moderna y visualmente atractiva, los usuarios pueden explorar:

- **Etimología**: Descomposición del nombre en sus componentes griegos (Τρίς + Ποῦς)
- **Historia**: Línea de tiempo desde la antigua Grecia hasta la actualidad
- **Diáspora**: Migración y distribución geográfica del apellido
- **Heráldica**: Escudos de armas y simbolismo familiar
- **Hipótesis**: Teorías sobre el origen y significado del apellido
- **Personajes notables**: Individuos destacados con el apellido Tripodi

## 🎨 Características Técnicas

- **Diseño responsive** con temas claro/oscuro
- **Animaciones suaves** y efectos visuales

## 🚀 Cómo Usar

1. **Abrir el sitio**: Simplemente abre `index.html` en cualquier navegador moderno
2. **Navegación**: Usa el menú superior para saltar a diferentes secciones
3. **Tema**: Cambia entre modo claro/oscuro usando el botón en la esquina superior derecha
4. **Interactividad**: Haz clic en imágenes para ampliarlas, explora la línea de tiempo y escucha pronunciaciones

## 🎯 Secciones Principales

### 1. **Etimología**
Desglose lingüístico del nombre:
- **Τρίς** (Trís) = Tres veces
- **Ποῦς** (Poûs) = Pie
- **Τρίποδι** (Trípodi) = Tripode (tres pies)

### 2. **Línea de Tiempo**
Cronología histórica desde:
- **Siglo VIII a.C.**: Primeras menciones en la antigua Grecia
- **Edad Media**: Migración a Italia
- **Siglo XIX**: Diáspora a América
- **Actualidad**: Distribución global

### 3. **Distribución Geográfica**
Mapas y estadísticas sobre la presencia del apellido en:
- **Italia** (Calabria, Sicilia)
- **América** (Argentina, Estados Unidos, Brasil)
- **Europa** (Francia, Alemania)

### 4. **Heráldica**
Escudos de armas y simbolismo:
- **Tripodi de Calabria**: León rampante
- **Tripodi de Sicilia**: Águila imperial
